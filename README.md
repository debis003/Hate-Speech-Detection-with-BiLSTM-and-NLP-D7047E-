# Advanced Deep Learning — D7047E

Coursework from **Advanced Deep Learning (D7047E)** at Luleå University of Technology (2025).

This course covered modern deep learning techniques through hands-on labs and a group project. The main deliverables included:

- **Hate Speech Detection (Group Project)** — Fine-tuned a pretrained RoBERTa transformer (`cardiffnlp/twitter-roberta-base`) on the HASOC 2019 dataset (~9,000 posts) for hate speech classification. Implemented adversarial training via Fast Gradient Method (FGM) and attention-based explainability. Achieved 0.87 accuracy and 0.86 F1.
- **Image Captioning** — Built a CNN encoder (ResNet) + LSTM decoder with attention mechanism for automatic image caption generation.
- **GAN & Adversarial Attacks** — Implemented a GAN for MNIST digit generation and gradient-based adversarial attacks demonstrating neural network vulnerabilities.
- **Additional labs** on neural network fundamentals, backpropagation, and optimization.

The source code for the group project and labs was developed collaboratively in [williamporomaa/D7047E_exercise_group_5](https://github.com/williamporomaa/D7047E_exercise_group_5). This repository contains the final reports and presentations.

**Technologies**: Python, PyTorch, Hugging Face Transformers, RoBERTa, GANs, ResNet, LSTM, Matplotlib
